# Equity Research Data Automation Dashboard (Python + Excel)

This project automates the sourcing, cleaning, merging, and visualization of
**equity-market** and **macro-economic** data to support research and investment
teams. It produces a **research-ready merged dataset** and an **Excel dashboard**
with filters and anomaly flags for quick analysis.

This is designed as a portfolio project for research/modeling roles in global
investment banks (e.g., UBS Interactive Modeling Analyst).
